⏱️ Stopwatch Application - User Guide

🎯 Overview
This Stopwatch application provides an intuitive interface to measure time intervals, record laps, and save session data.

🖥️ Interface Overview
- Time Display → Shows elapsed time in `HH:MM:SS` format.
- Start Button → Begins the stopwatch.
- Stop Button → Pauses the stopwatch.
- Reset Button → Resets the stopwatch to `00:00:00`.
- Lap Button → Records a lap time.
- Lap Display → Shows all recorded lap times.

🏃‍♂️ How to Use
▶️ Start Stopwatch
- Click the "Start" button.
- You can also press the `Spacebar` key to start.

⏹️ Stop Stopwatch
- Click the "Stop" button to pause.
- Press the `Spacebar` key again to stop.

🔄 Reset Stopwatch
- Click the "Reset" button to reset time to `00:00:00`.
- Press the `R` key to reset.

🏁 Record Laps
- Click the "Lap" button to record the current time.
- Press the `L` key to record a lap.

🌗 Toggle Dark/Light Mode
- Click the "Toggle Theme" button to switch between dark and light mode.

🎹 Keyboard Shortcuts
| Action | Shortcut Key |
|--------|-------------|
| Start / Stop | `Spacebar` |
| Reset | `R` |
| Record Lap | `L` |
| Toggle Dark Mode | `T` |

📌 Example Usage
1. Start the stopwatch.
2. Press `L` to record a lap at any point.
3. Stop the stopwatch when done.
4. Reset if you want to start a new session.

For advanced usage, visit the [GitHub repository](https://github.com/kabilesh-c/Stopwatch-Project).
